Philosophy

Traditional UIs are built on clicks - binary, instant decisions. Eddystrap is different.

We use time as input. Holding builds commitment. Duration matters. Everything flows with kinetic feedback and natural metaphors.

No traditional buttons. Just natural interaction.

Quick Start

npm install eddystrap
import { Eddy, Pool } from 'eddystrap';
import 'eddystrap/styles.css';

const eddy = new Eddy({
  color: '#6366f1',
  label: 'Confirm',
  onActivate: () => console.log('Committed!')
});

eddy.mount('#app');
