Eddy

Stable

The core component. Press and hold to build commitment. Commitment builds over time through kinetic feedback - ripples, glow, and a progress circle. Release early and nothing happens. Hold until completion and the action triggers.

Pool

Stable

A modal alternative using natural flow metaphors. Instead of "open/close", you "enter" a pool and "let it drain". The UI flows in with ripple animations rather than popping into existence.

const pool = new Pool({ color: '#6366f1', content: '<h2>Welcome</h2><p>Content here</p>', onDrain: () => console.log('Drained') }); pool.mount('body'); pool.enter(); // Show it

Stream

Experimental

Text input with flowing ripple effects. As you type, ripples emanate from random positions, creating a sense of text flowing into the field. Still finding its identity beyond being a decorated input.

Type to see it flow...
const stream = new Stream({ color: '#8b5cf6', placeholder: 'Type your thoughts...', multiline: false, onChange: (value) => console.log(value) });

Wave

Experimental

Press and hold to make the wave grow. Release when it reaches the desired amplitude. The wave naturally decays over time. No dragging - time builds the wave, just like holding water raises the level.

Tide

Experimental

Toggle with inertia. Click works instantly (practical), but the tide rises and falls gradually with momentum. The visual transition takes ~800ms, creating a weighted, natural feel without being annoying.
